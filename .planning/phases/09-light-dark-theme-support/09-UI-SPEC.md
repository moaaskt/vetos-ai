## Color

| Role | Value (Light Mode) | Value (Dark Mode) | Usage |
|------|--------------------|-------------------|-------|
| Dominant (60%) | `oklch(0.98 0.005 240)` | `oklch(0.18 0.012 235)` | Background, main application surfaces |
| Secondary (30%) | `oklch(1 0 0)` | `oklch(0.22 0.012 230)` | Cards, sidebars, modals, dropdowns |
| Accent (10%) | `oklch(0.62 0.07 183)` | `oklch(0.62 0.07 183)` | Primary action buttons, active navigation indicators, theme toggle active state |
| Destructive | `oklch(0.64 0.14 25)` | `oklch(0.64 0.14 25)` | Delete buttons, error badges |

Accent reserved for: Theme toggle interactive icon highlight, primary action buttons, active navigation indicators, active tab borders.
