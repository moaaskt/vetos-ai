## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `slate-950` | Background, main application surfaces |
| Secondary (30%) | `slate-900` | Cards, sidebars, modals, empty state backgrounds |
| Accent (10%) | `teal-400` | Primary buttons, active navigation indicators, key metrics |
| Destructive | `red-500` | Delete buttons, error badges, impersonation exit banners |

Accent reserved for: Primary action buttons, active navigation states, selected focus rings, key success metric highlights.

---

## Animation & Micro-interactions

| State/Action | Behavior |
|--------------|----------|
| Hover | Subtle background lightness shift (`hover:bg-white/5`), card elevation (`shadow-lg`). |
| Focus | Visible ring (`focus:ring-2 focus:ring-teal-400/50 focus:outline-none`). |
| Loading | Premium skeleton states (pulsing `bg-slate-800`). |
| Transitions | Smooth modal fades (`duration-200 ease-out`), soft page mounting. |

---

## Copywriting Contract

| Element | Copy |
|---------|------|
| Primary CTA | Action-oriented (e.g., `Save Changes`, `Schedule Appointment`) |
| Empty state heading | Contextual and friendly (e.g., `No appointments scheduled today.`) |
| Empty state body | Humanized direction (e.g., `When you book a visit, it will appear here. [Book Appointment]`) |
| Error state | Actionable (e.g., `Unable to load data. Please refresh the page.`) |
| Destructive confirmation | Explicit risk (e.g., `Delete Clinic: Are you sure you want to permanently remove this clinic?`) |

---

## Empty States & Feedback

- **Visuals:** Custom SVG illustrations tailored to veterinary operations (pets, appointments). Clean and minimalistic, avoiding generic dashboard graphics.
- **Tenant Feel:** Warm, welcoming, operation-oriented (focused on daily tasks).
- **Super Admin Feel:** Analytical, platform-oriented, focused on metrics and management.
