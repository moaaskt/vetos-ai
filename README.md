# VetOS AI

A scalable SaaS platform for veterinary clinics and petshops with WhatsApp integration and AI-powered assistance. The system centralizes customer communication, appointment scheduling, pet records, and intelligent automation.

## Project Structure

This project is a monorepo consisting of:

- `backend/`: NestJS backend API.
- `frontend/`: React (Vite) + TailwindCSS frontend application.
- `docker-compose.yml`: Local infrastructure setup (PostgreSQL + Redis).
- `.planning/`: GSD workflow planning documents (PROJECT.md, REQUIREMENTS.md, ROADMAP.md).

## Prerequisites

- Node.js (v18+)
- Docker & Docker Compose
- npm or yarn

## Getting Started

### 1. Start Infrastructure

Start the PostgreSQL and Redis containers:

```bash
docker-compose up -d
```

### 2. Setup Backend

Navigate to the `backend` directory and start the server:

```bash
cd backend
npm install
npm run start:dev
```

### 3. Setup Frontend

Navigate to the `frontend` directory and start the development server:

```bash
cd frontend
npm install
npm run dev
```

## Technologies Used

- **Backend**: NestJS, Node.js, Prisma/TypeORM (planned), Redis
- **Frontend**: React, Vite, TailwindCSS
- **Database**: PostgreSQL
- **Integrations**: WhatsApp Cloud API, OpenAI/Gemini

## Architecture Highlights

- **Modular Architecture**: Built with Domain-Driven Design style via NestJS modules.
- **Multi-Tenant**: Designed to isolate data per clinic/petshop.
- **Queue System**: Redis-based async processing for WhatsApp messaging and AI calls.
