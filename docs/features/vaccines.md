# Feature: Vaccines & Protocols

Gerenciamento de esquemas de vacinação, regras de doses e histórico vacinal.

## Descrição Geral
Permite que cada clínica crie protocolos vacinais personalizados (ex: esquema V10/Quádrupla Felina), estabelecendo intervalos em dias entre as doses e agendando automaticamente os próximos disparos vacinais.
